# ACSD-68892: Inkonsistentes [!DNL Fastly]-Caching für zwischenspeicherbare Seiten

Der Patch ACSD-68892 behebt das Problem, dass zwischenspeicherbare Seiten nicht ordnungsgemäß gespeichert oder aus dem [!DNL Fastly]-Cache bereitgestellt werden, was zu inkonsistentem Caching-Verhalten und reduzierter Leistung führt. Dieser Patch ist verfügbar, wenn [[!DNL Quality Patches Tool (QPT)]](/help/tools/quality-patches-tool/quality-patches-tool-to-self-serve-quality-patches.md) 1.1.77 installiert ist. Die Patch-ID ist ACSD-68892. Dieses Problem wird voraussichtlich in Adobe Commerce 2.4.9 behoben.

## Betroffene Produkte und Versionen

**Der Patch wird für die Adobe Commerce-Version erstellt:**

* Adobe Commerce (alle Bereitstellungsmethoden) 2.4.8-p2

**Kompatibel mit Adobe Commerce-Versionen:**

* Adobe Commerce (alle Bereitstellungsmethoden) 2.4.8 - 2.4.8-p3

>[!NOTE]
>
>Der Patch könnte mit neuen [!DNL Quality Patches Tool]-Versionen auch für andere Versionen gelten. Um zu überprüfen, ob der Patch mit Ihrer Adobe Commerce-Version kompatibel ist, aktualisieren Sie das `magento/quality-patches` auf die neueste Version und überprüfen Sie die Kompatibilität auf der Seite [[!DNL Quality Patches Tool]: Nach Patches suchen](https://experienceleague.adobe.com/tools/commerce-quality-patches/index.html). Verwenden Sie die Patch-ID als Suchbegriff, um den Patch zu finden.

## Problem

Zwischenspeicherbare Seiten werden nicht ordnungsgemäß gespeichert oder aus dem [!DNL Fastly]-Cache bereitgestellt, was zu einem inkonsistenten Caching-Verhalten und geringerer Leistung führt.

<u>Schritte zur Reproduktion</u>:

1. Aktivieren Sie den [!DNL Fastly].
1. Erstellen Sie eine neue Website/Store/Store-Ansicht.
1. Fügen Sie den sekundären Website-Code zu `index.php` hinzu:

```
$_SERVER['MAGE_RUN_TYPE'] = 'website';
$_SERVER['MAGE_RUN_CODE'] = 'web2';
```

1. Cache-Steuerungs-Header mit cURL überprüfen:

```
curl -Is https://localhost/m247/pub/ | grep cache-control
```

<u>Erwartete Ergebnisse</u>:

Zwischenspeicherbare Seiten werden in [!DNL Fastly] zwischengespeichert.

<u>Tatsächliche Ergebnisse</u>:

Zwischenspeicherbare Seiten werden nicht in [!DNL Fastly] zwischengespeichert, die in der -Kopfzeile zurückgegeben `max-age=0`.

```
curl -Is https://localhost/m24develop/pub/ |grep cache-control
cache-control: max-age=0, must-revalidate, no-cache, no-store
```

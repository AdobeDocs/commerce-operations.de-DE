# ACSD-69311: Falsche Steuerberechnung in Gutschriften nach Teilerstattungen aus Rechnungen

Der Patch ACSD-69311 behebt das Problem, dass bei falschen Steuerberechnungen in Gutschriften beim Erstellen einer Teilerstattung aus einer Rechnung eine vorherige Gutschrift auf der Bestellansichtsseite erstellt wurde. Dieser Patch ist verfügbar, wenn [[!DNL Quality Patches Tool (QPT)]](/help/tools/quality-patches-tool/quality-patches-tool-to-self-serve-quality-patches.md) 1.1.77 installiert ist. Die Patch-ID ist ACSD-69311. Dieses Problem wird voraussichtlich in Adobe Commerce 2.4.9 behoben.

## Betroffene Produkte und Versionen

**Der Patch wird für die Adobe Commerce-Version erstellt:**

* Adobe Commerce (alle Bereitstellungsmethoden) 2.4.7-p8

**Kompatibel mit Adobe Commerce-Versionen:**

* Adobe Commerce (alle Bereitstellungsmethoden) 2.4.4 - 2.4.8-p3

>[!NOTE]
>
>Der Patch könnte mit neuen [!DNL Quality Patches Tool]-Versionen auch für andere Versionen gelten. Um zu überprüfen, ob der Patch mit Ihrer Adobe Commerce-Version kompatibel ist, aktualisieren Sie das `magento/quality-patches` auf die neueste Version und überprüfen Sie die Kompatibilität auf der Seite [[!DNL Quality Patches Tool]: Nach Patches suchen](https://experienceleague.adobe.com/tools/commerce-quality-patches/index.html). Verwenden Sie die Patch-ID als Suchbegriff, um den Patch zu finden.

## Problem

Fehlerhafte Steuerberechnungen treten bei der Erstellung einer Teilerstattung aus einer Rechnung in Gutschriften auf, wenn eine vorherige Gutschrift auf der Seite „Auftragsansicht“ erstellt wurde.

<u>Schritte zur Reproduktion</u>:

1. Konfigurieren Sie Katalogpreise, um Steuern einzuschließen.
1. Alle angezeigten Preise so konfigurieren, dass Steuern enthalten sind.
1. Erstellen Sie einen Steuersatz und eine Steuerregel für *10 %* für *Australien*.
1. Erstellen Sie 2 Produkte mit Preisen *100* und *200*.
1. Fügen Sie beide Produkte zum Warenkorb hinzu.
1. Probieren Sie es aus, indem Sie einen kostenlosen Versand und eine australische Adresse verwenden.
1. Öffnen Sie die Bestellung im [!UICONTROL Admin].
1. Rechnung für die Bestellung erstellen.
1. Erstellen Sie eine Teilgutschrift über die Schaltfläche auf der Seite „Bestellansicht“, indem Sie ein Produkt auswählen.
1. Öffnen Sie die Rechnung.
1. Erstellen Sie eine Gutschrift über die Schaltfläche in der Rechnung.

<u>Erwartete Ergebnisse</u>:

Der Steuerbetrag wird in der Gutschrift für die Rückerstattung korrekt wiedergegeben.

<u>Tatsächliche Ergebnisse</u>:

Die zweite Gutschrift enthält den vollen Steuerbetrag in der Summe für die Rückerstattung.

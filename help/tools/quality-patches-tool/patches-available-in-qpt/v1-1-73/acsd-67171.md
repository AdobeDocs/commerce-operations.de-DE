# ACSD-67171: Abgelaufene [!DNL B2B]-Sitzung während des Checkouts leitet zur Seite Zugriff verweigert anstelle der Anmeldung weiter

Mit dem Patch „ACSD-67171“ wird ein Problem behoben, bei dem [!DNL B2B] Kunden zu einer Seite *Zugriff verweigert* weitergeleitet werden, wenn die Sitzung abgelaufen ist oder während des Checkouts entfernt wird. Dieser Patch ist verfügbar, wenn [[!DNL Quality Patches Tool (QPT)]](/help/tools/quality-patches-tool/quality-patches-tool-to-self-serve-quality-patches.md) 1.1.73 installiert ist. Die Patch-ID ist ACSD-67171. Dieses Problem wird voraussichtlich in Adobe Commerce 2.4.9 behoben.

## Betroffene Produkte und Versionen

**Der Patch wird für die Adobe Commerce-Version erstellt:**

* Adobe Commerce (alle Bereitstellungsmethoden) 2.4.8-p1

**Kompatibel mit Adobe Commerce-Versionen:**

* Adobe Commerce (alle Bereitstellungsmethoden) 2.4.4 - 2.4.8-p3

>[!NOTE]
>
>Der Patch könnte mit neuen [!DNL Quality Patches Tool]-Versionen auch für andere Versionen gelten. Um zu überprüfen, ob der Patch mit Ihrer Adobe Commerce-Version kompatibel ist, aktualisieren Sie das `magento/quality-patches` auf die neueste Version und überprüfen Sie die Kompatibilität auf der Seite [[!DNL Quality Patches Tool]: Nach Patches suchen](https://experienceleague.adobe.com/tools/commerce-quality-patches/index.html). Verwenden Sie die Patch-ID als Suchbegriff, um den Patch zu finden.

## Problem

Das System zeigt während des Auscheckens eine Seite Zugriff verweigert an, wenn die Sitzung eines [!DNL B2B] Benutzers abläuft oder entfernt wird, anstatt ihn zur Anmeldeseite des Kunden weiterzuleiten.

<u>Schritte zur Reproduktion</u>:

1. Gehen Sie im Admin-Bedienfeld zu [!UICONTROL Stores] > [!UICONTROL Configuration] > [!UICONTROL General] > [!UICONTROL [!DNL B2B] Features], legen Sie [!UICONTROL [!DNL B2B] Features] auf *Ja* fest, gehen Sie dann zu [!UICONTROL Customers] > [!UICONTROL Persistent Shopping Cart] und setzen Sie [!UICONTROL Enable Persistence] auf *Ja*.
1. Gehen Sie zu [!UICONTROL Customers] > [!UICONTROL Companies] und wählen Sie [!UICONTROL Add New Company] aus, um eine neue [!DNL B2B] Firma zu erstellen.
1. Melden Sie sich bei der Storefront als [!DNL B2B] Unternehmensbenutzer an.
1. Fügen Sie ein beliebiges Produkt zum Warenkorb hinzu.
1. Wechseln Sie zur Seite Checkout .
1. Die Sitzung darf ablaufen.
1. Laden Sie die Checkout-Seite neu.

<u>Erwartete Ergebnisse</u>:

Sie werden zur Anmeldeseite von Kunden weitergeleitet.

<u>Tatsächliche Ergebnisse</u>:

Sie werden zu `/company/accessdenied` weitergeleitet und sehen *Sie haben keine Berechtigung, diese Seite anzuzeigen. Wenn Sie glauben, dass dies ein Fehler ist, wenden Sie sich an Ihren Unternehmensadministrator.*

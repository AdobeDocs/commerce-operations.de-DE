# ACP2E-4416: Kundenprämienpunkte werden nicht initialisiert, wenn sie in der Admin erstellt werden

Mit dem Patch ACP2E-4416 wird das Problem behoben, dass Kundenbelohnungspunkte nicht initialisiert werden, wenn sie in der Admin erstellt wurden. Dieser Patch ist verfügbar, wenn [[!DNL Quality Patches Tool (QPT)]](/help/tools/quality-patches-tool/quality-patches-tool-to-self-serve-quality-patches.md) 1.1.78 installiert ist. Die Patch-ID lautet ACP2E-4416. Dieses Problem wird voraussichtlich in Adobe Commerce 2.4.9 behoben.

## Betroffene Produkte und Versionen

**Der Patch wird für die Adobe Commerce-Version erstellt:**

* Adobe Commerce (alle Bereitstellungsmethoden) 2.4.8-p3

**Kompatibel mit Adobe Commerce-Versionen:**

* Adobe Commerce (alle Bereitstellungsmethoden) 2.4.4 - 2.4.8-p4

>[!NOTE]
>
>Der Patch könnte mit neuen [!DNL Quality Patches Tool]-Versionen auch für andere Versionen gelten. Um zu überprüfen, ob der Patch mit Ihrer Adobe Commerce-Version kompatibel ist, aktualisieren Sie das `magento/quality-patches` auf die neueste Version und überprüfen Sie die Kompatibilität auf der Seite [[!DNL Quality Patches Tool]: Nach Patches suchen](https://experienceleague.adobe.com/tools/commerce-quality-patches/index.html?lang=de). Verwenden Sie die Patch-ID als Suchbegriff, um den Patch zu finden.

## Problem

Kundenbelohnungspunkte werden nicht initialisiert, wenn sie im Administrator erstellt werden.

<u>Schritte zur Reproduktion</u>:

1. Erstellen Sie einen Kunden über die Admin- oder die REST-API.
1. Senden Sie eine GraphQL-Anfrage, um den Prämiensaldo des Kunden abzurufen.
1. Überprüfen Sie die Antwort.

<u>Erwartete Ergebnisse</u>:

Das System gibt den Prämienpunkt-Saldo des Kunden zurück.

<u>Tatsächliche Ergebnisse</u>:

Das System gibt einen Fehler zurück:

```
{"errors":[{"message":"Internal server error","locations":[{"line":5,"column":9}],"path":["customer","reward_points","balance","points"]}],"data":{"customer":{"reward_points":{"balance":null}}}}
```

## Patch anwenden

Verwenden Sie je nach Bereitstellungsmethode die folgenden Links, um einzelne Patches anzuwenden:

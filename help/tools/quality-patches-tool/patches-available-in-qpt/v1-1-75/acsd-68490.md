# ACSD-68490: **[!UICONTROL Add New Attribute]** Schaltfläche für Administratoren mit eingeschränkter Administratorberechtigung während der konfigurierbaren Produkterstellung sichtbar

Mit dem Patch ACSD-68490 wird das Problem behoben, dass **[!UICONTROL Add New Attribute]** Schaltfläche während der konfigurierbaren Produkterstellung für einen Administrator mit eingeschränkter Zugriffsberechtigung sichtbar ist. Dieser Patch ist verfügbar, wenn [[!DNL Quality Patches Tool (QPT)]](/help/tools/quality-patches-tool/quality-patches-tool-to-self-serve-quality-patches.md) 1.1.75 installiert ist. Die Patch-ID ist ACSD-68490. Dieses Problem wird voraussichtlich in Adobe Commerce 2.4.9 behoben.

## Betroffene Produkte und Versionen

**Der Patch wird für die Adobe Commerce-Version erstellt:**

* Adobe Commerce (alle Bereitstellungsmethoden) 2.4.6

**Kompatibel mit Adobe Commerce-Versionen:**

* Adobe Commerce (alle Bereitstellungsmethoden) 2.4.6 - 2.4.6-p13

>[!NOTE]
>
>Der Patch könnte mit neuen [!DNL Quality Patches Tool]-Versionen auch für andere Versionen gelten. Um zu überprüfen, ob der Patch mit Ihrer Adobe Commerce-Version kompatibel ist, aktualisieren Sie das `magento/quality-patches` auf die neueste Version und überprüfen Sie die Kompatibilität auf der Seite [[!DNL Quality Patches Tool]: Nach Patches suchen](https://experienceleague.adobe.com/tools/commerce-quality-patches/index.html). Verwenden Sie die Patch-ID als Suchbegriff, um den Patch zu finden.

## Problem

Die Schaltfläche &quot;**[!UICONTROL Add New Attribute]**&quot; ist während der konfigurierbaren Produkterstellung für Administratoren mit eingeschränkter Zugriffsberechtigung sichtbar.

<u>Schritte zur Reproduktion</u>:

1. Erstellen Sie eine zusätzliche Website-, Store- und Store-Ansicht.
1. Erstellen Sie eine Administratorrolle, die nur Zugriff auf die neue Website hat, und weisen Sie alle Ressourcen zu.
1. Erstellen Sie einen eingeschränkten Admin-Benutzer und weisen Sie diesem Benutzer die neue Rolle zu.
1. Melden Sie sich bei Admin als eingeschränkter Benutzer an.
1. Erstellen Sie ein neues konfigurierbares Produkt.


<u>Erwartete Ergebnisse</u>:

Die Schaltfläche &quot;**[!UICONTROL Create New Attribute]**&quot; ist für Administratoren mit beschränkten Rechten nicht verfügbar.

<u>Tatsächliche Ergebnisse</u>:

Eine 404-Seite wird angezeigt.

Verwenden Sie je nach Bereitstellungsmethode die folgenden Links, um einzelne Patches anzuwenden:
